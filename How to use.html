Out of lack of Jquery hovercard plugins, I decided to make one. Simple to use, and pretty customisable.
To view usage and docs, visit this page

Basic usage:
Add Jquery to the head of your site, followed by this plugin. To use it, simply put this at the bottom of your page:

<script>
$(".hovercard").tghovercard({
delayShow: "300",
delayHide: "300",
height: "150",
fadeOut: "300",
fadeIn: "300",
defaultcss: "yes",
url: "/hovercards.php?user=",
top: "30",
menuHeight: "40"
});
</script>
To get data from a remote file, change the option "url" value to the file that gets the remote data. Using a GET variable at the end (like the default setting for url uses) allows you to load different data, depending on a data attribute in your link.
For example, in this link, you can see "data-uid" has a value of "1". The plugin will attach that value to the end of the "url" option. The page/script you link to can access the GET variable and post back any information based on the variable
<a href="/index" class="hovercard" data-uid="1">User 1</a>
Visit the website for a detailed example on usage.
